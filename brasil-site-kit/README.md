# Brasil Bistro — Kit visual para site

## O que está neste pacote
- `logo/`: logos prontos para web em PNG transparente
- `icons/`: ícones extraídos do PDF
- `branding/tokens.css`: variáveis CSS da marca
- `branding/tokens.json`: tokens em JSON
- `sample/site-header-example.html`: exemplo pronto para usar

## Paleta principal
- Dourado: `#DBAA09`
- Preto: `#000000`
- Branco: `#FFFFFF`

## Observações
- O PDF mostra o wordmark e os símbolos, mas não inclui a fonte original do logo como arquivo separado.
- Por isso, o kit entrega os logos já exportados em PNG transparente, prontos para uso no site.
- Se você quiser, depois posso transformar isso em um layout completo de landing page.
