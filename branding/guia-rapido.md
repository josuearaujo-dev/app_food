## O que está neste pacote
- `logo/logo-principal-transparent.png` — logo principal com fundo transparente
- `logo/logo-principal-bg-nude.png` — logo em fundo nude
- `logo/logo-perfil-1024.png` — versão quadrada para perfil
- `logo/favicon-512.png` e `logo/favicon-192.png` — favicons
- `branding/tokens.css` — variáveis CSS prontas para o site
- `branding/tokens.json` — tokens para integrar em projetos React, Next, Vue ou CMS
- `sample/site-header-example.html` — exemplo simples de uso

## Cores oficiais
- Marrom: `#531B04`
- Nude: `#E1D3C7`
- Rosa: `#FB5B77`

## Tipografia indicada no material
- Principal: `Ag Adumu Regular`
- Secundária: `Proxima Nova`

## Observação importante
As fontes não acompanham este pacote. Para usar no site, você precisa licenciar/instalar as fontes ou substituir por equivalentes.

## Recomendações para o site
- Use o nude como fundo principal
- Use o marrom para texto, botões e títulos
- Use o rosa para destaques, badges e CTAs secundários
- Preserve área de respiro ao redor do logo
- Prefira aplicar o logo principal em fundos claros
